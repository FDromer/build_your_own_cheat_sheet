CSS Selectors Table Reference

Selectors Example Description
Type p It's matches the type of the element in the HTML document.
Universal * It's selects all elements of any type.
Class .brand It selects elements with a specific class attribute.
id #large-title It matches an element based on the value of the element's id attribute.
Attribute a[href*='florence'] It's used to select an element with some specific attribute or attribute value.
Pseudo-class p:hover It's a keyword added to a selector that specifies a special state of the selected element(s).
Chaining h1.special It's a combination of multiple selectors which will be affected by the same effect.
Descendant Combinator .main-list li combines two selectors such that elements matched by the second selector are selected if they have an ancestor element matching the first selector.
Multiple Selectors h1, .menu It's group multiple selectors that share the same declaration.
